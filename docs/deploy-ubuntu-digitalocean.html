Deploy an Ubuntu 18.04 Linux Distribution on Digital Ocean

Prerequisites

This guide requires a DigitalOcean account.

Create New Account on Digital Ocean


After you log into your DigitalOcean account for the first time you need to setup an SSH key on your MacOS computer.

Setup an SSH Key on MacOS to Connect to your Digital Ocean Droplet

Create Project

    1. Click on New Project in the left side menu
    1. Enter project name
    1. Enter project description
    1. Select Web Application from drop down menu
    1. Click Create Project button to finalize

Move Resources

You can move resources from other projects to this new one, but we will skip this step for now.

    1. Click Skip for now

Create Droplet

    1. Click Get Started with a Droplet

Ubuntu Distribution

    1. Click down arrow on the Ubuntu symbol
    1. Select the 18.04 x64 from the dropdown menu

Plan & Backups

    1. Choose Standard plan
    1. Select the plan that fits your needs and budget
    1. Add Backups (optional)

Data Center

    1. Choose the Data Center Region closest to you current geographical location

SSH Key

This will allow you to use the Terminal to remote login from your Macbook Pro.

Remote login is required to setup Elasticsearch, deploy Node / React apps.

It is also required to Setup Secure HTTPS/SSL Reverse Proxy with Nginx

    1. Click on the New SSH Key button

Add SSH Key - Macbook Pro

    1. In the Terminal use the command below to cd into the .ssh directory
cd ~/.ssh
    1. Use the command below to Copy the key to to the Terminal output
cat id_rsa.pub
    1. Select the key and hit Cmd + C to Copy the key to the clipboard
    1. Use Cmd + V to Paste the key into the SSH key content box
    1. Name the key id_rsa_pub_macbook_pro
    1. Click the Add SSH Key button to finalize


This key can now be added to future distributions


Finalize and Deploy

    1. Choose a name
    1. Click the Create button to deploy

View Your New Ubuntu 18.04 Linux Distribution

    1. When the deployment is complete click on the link to view your new distribution

Enable Private IP

    1. Click on the Networking link
    1. Click on the Enable now link next to Private IP
    1. When the Private IP has been enbaled click on the IP ADRESS to Copy the IP to the clipboard

Remote Login

  1. In the Terminal use the command below to Remote Login into your new Ubuntu 18.04 Linux Distribution

NOTE: Use your Private IP copied from the previous step

ssh root@YOUR_IP_ADDRESS
You will be asked to verify connection.
